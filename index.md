# 使用navicat操作mysql数据库注意事项

本片主要记录工具-->命令行工具 中使用的指令操作

## 查看数据库
### `show databases;`(注意后面一定要带分号)

## 使用相关数据库
### `use test;`

## 新建数据库
### `create database xxx;`

## 删除一个数据库
### `drop database xxx;`

## 查看表的结构
### `desc xxx;`

## 新建表
### `create table xxx (a BIGINT , b VARCHAR(20),c INT);`

## 删除表
### `drop table student;`
